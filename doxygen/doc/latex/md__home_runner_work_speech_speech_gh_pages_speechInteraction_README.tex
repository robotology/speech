This folder includes different modules which use Google Cloud Services.
\begin{DoxyItemize}
\item {\bfseries{google\+Speech\+Process}}\+: it reveals the structure and the syntax of your textual or vocal input by analyzing also the sentiments of your phrases;
\item {\bfseries{google\+Speech}}\+: it performs speech-\/to-\/text;
\item {\bfseries{google\+Synthesis}}\+: it performs text-\/to-\/speech;
\item {\bfseries{google\+Dialog}}\+: it allows to carry on a conversation about a specific theme, corresponding to your own dialog flow created in your Google project.
\end{DoxyItemize}

In order to use them, the user have to register, create and download a {\itshape private key} from Google Cloud. Below the instructions on how to do\+:
\begin{DoxyEnumerate}
\item Browse to \href{https://console.cloud.google.com}{\texttt{ Cloud Console}} and create an account. This requires the user to insert a credit card\+: this process is mandatory in order to run the application, but you will not pay anything
\item Starting from the Dashboard page, under the Project info section, use the button {\bfseries{Go to Project Settings}} which will lead you to the Settings
\item Browse to {\bfseries{Service Accounts}} using the left navigation bar
\item In the table you will find all the service accounts active for your projects. Use the {\bfseries{Actions}} button related to the {\itshape dialog flow service} and select {\bfseries{Create key}} choosing the JSON format.
\end{DoxyEnumerate}\hypertarget{md__home_runner_work_speech_speech_gh_pages_speechInteraction_README_autotoc_md1}{}\doxysubsection{Some useful information for the user\+:}\label{md__home_runner_work_speech_speech_gh_pages_speechInteraction_README_autotoc_md1}
