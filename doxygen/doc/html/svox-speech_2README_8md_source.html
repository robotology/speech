3 A simple text to speech using **`SVOXPICO`**.

6 Requirements
7 ------------
8 The current implementation requires Linux **`aplay`** command (in the `alsa-utils` package)
9 to play the generated wave file. This can/should be easily replaced with the
10 `yarp portaudio` device (audio player).
11 
12 On Windows systems `powershell` is employed to play the wave file.

14 
15 Installation
16 ------------
17 - `$ cd svox-speech`
18 - `$ mkdir build && cd build`
19 - `$ cmake ..`
20 - tick on the driver `speech`.
21 - select a convenient location for installation via `CMAKE_INSTALL_PREFIX` (refer to this location as `$INST_DIR`).
22 - `$ make install`
23 - append to the environment variable **`YARP_DATA_DIRS`** the path `$INST_DIR/share/speech`.
